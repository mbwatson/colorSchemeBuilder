Color Scheme Builder

＋ COLOR

Change the RGB levels of a color block by scrolling over the pop-up color faders.

Add a new color block with the ＋ Color button.

Hovering on a color block yields a few opportunities:

  • Duplicate the color block with the icon;
  • Delete the color block with the icon; and
  • Copy the hexadecimal color code to your clipboard by clicking it.
